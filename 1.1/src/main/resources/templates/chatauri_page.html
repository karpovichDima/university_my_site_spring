What you need to know about this application

Purpose

Standard modern chat with cloud sync. The main feature of the application - it is possible to call. Moreover, it is possible to call even at low Internet speeds, which will clearly be an advantage when going out of town. The emphasis in the application is on speed and simplicity.

Design

The design is incredibly minimalist, so creative people will definitely like it. Since the feature of the application is simplicity, a special approach was required to the interface, for example, the user can change all the backgrounds in the application at his own discretion.
