What you need to know about this application

Purpose

If you need to know the weather, but go to the window too lazy, then this application is right for you. Displays information about the weather at the moment, the forecast for several days in advance. Very simple functionality for true connoisseurs of material design.

Design

The design is based on the concept of google, called "material", it gives the opportunity to give the user a greater interface response to any of its actions. It was used a lot of colors from a completely different palette. But there is not a single bright color, it turned out something very non-standard and rather restrained.
